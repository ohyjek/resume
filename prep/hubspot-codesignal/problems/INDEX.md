# HubSpot Problem Library Index

This index tracks all seeded problems, status, and canonical paths.

## Status legend

- `Queued`: ready for pipeline dispatch
- `InProgress`: currently running
- `Review`: artifacts generated and pending human validation
- `Done`: reviewed and ready for reuse
- `Blocked`: pipeline failed and requires intervention

## Workflow note

This file is a catalog/index. The active execution flow is manual fan-out from per-problem
`agent-prompt.md` files under `artifacts/...`.
