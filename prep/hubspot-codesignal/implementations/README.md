# Implementations

This folder stores concrete outputs generated by the artifact skill, separate from `problems/`.

## Layout

- `dsa/<problem-slug>.ts`: implementation-ready TypeScript for DSA problems
- `system-design/<problem-slug>.md`: implementation-oriented notes for system design problems

## Source of truth

Each implementation maps back to its artifact bundle:

- `prep/hubspot-codesignal/artifacts/<category>/<problem-slug>/solution.md`
- `prep/hubspot-codesignal/artifacts/<category>/<problem-slug>/reasoning.md`
- category teaching artifact (`teaching-stub.ts` or `teaching-notes.md`)

## Workflow

1. Generate/reset artifact bundles and prompts.
2. Run `/hubspot-problem-agent` for one artifact directory.
3. Skill updates artifact files and writes implementation output here.
